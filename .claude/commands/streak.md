# 复盘 Streak 查询

查看复盘连续天数和统计信息。

## 前置检查

1. 读取 `trading-config.md` 确认已完成配置
2. 读取 `references/analysis-rules.md` 的 Streak 部分获取计算规则

## 执行步骤

1. 使用 Notion MCP 工具从复盘日历获取所有 类型=日复盘 的记录
2. 提取日期列表，按日期降序排列
3. 从今天（或最近一个交易日）开始往回数
4. 跳过周末（周六日）和美股休市日
5. 计算连续有复盘记录的交易日数量 = streak

## 输出格式

```
复盘 Streak: {N} 天
本月复盘率: {completed}/{total_trading_days} ({percent}%)
```

## 里程碑检测

如果 streak 达到里程碑（7/14/21/30/50/100/365天），显示对应的里程碑消息。

## 断链检测

如果昨天是交易日但没有复盘记录，显示断链提醒并鼓励重新开始。

## 用户输入

$ARGUMENTS
