# 每日复盘

执行结构化的每日交易复盘。

## 前置检查

1. 读取 `trading-config.md` 确认已完成配置
2. 读取 `references/notion-schema.md` 获取复盘日历 schema

## 复盘流程

1. 使用 Notion MCP 工具读取当日交易记录
2. 展示当日交易摘要和 P&L 汇总
3. 引导用户回答以下问题（逐步引导，不要一次全问）：
   - 今天执行了什么 setup？
   - 情绪状态如何？
   - 最佳交易是哪笔？为什么？
   - 最差交易是哪笔？为什么？
   - 明天有什么需要改进的？
4. 在复盘日历创建 "日复盘" 条目，关联当日交易记录
5. 用户回答写入页面正文
6. 更新并显示 streak 信息

## 复盘日历写入格式

参考 `references/notion-schema.md` 的复盘日历 schema：
- Name: "YYYY-MM-DD 日复盘"
- 类型: 日复盘
- 日期: 当天
- 关联当日交易记录

## 如果当天没有交易

仍然允许复盘，提示用户记录观察和心得。保持记录的习惯比内容完美更重要。

## 用户输入

$ARGUMENTS
