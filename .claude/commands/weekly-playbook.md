# Weekly Playbook 周度策略报告

生成周度策略复盘和下周计划。

## 前置检查

1. 读取 `trading-config.md` 确认已完成配置
2. 读取 `references/analysis-rules.md` 的 Weekly Playbook 部分获取完整规则
3. 读取 `references/notion-schema.md` 获取写入格式

## 数据收集

1. 使用 Notion MCP 工具获取本周一至今的所有交易记录
2. 获取上周交易记录（用于对比）
3. 获取本周复盘记录
4. Web search 下周重大经济数据和财报日历

## 报告结构

1. **本周总结** — 交易笔数、胜率、总P&L、平均R、最大单笔盈亏（对比上周）
2. **Best & Worst Trade** — 最佳和最差交易详情 + 分析
3. **错误统计** — 本周 vs 上周错误次数和成本
4. **策略表现** — 按策略的笔数、胜率、P&L
5. **复盘完成度** — 当前 streak + 本周复盘率
6. **下周关注** — 经济数据、重要财报、技术位
7. **行动项** — 基于本周分析的具体改进建议

## 写入 Notion

报告完成后，自动写入复盘日历：
- Name: "YYYY-MM-DD 周复盘"
- 类型: 周复盘
- 日期: 本周日
- 关联本周所有交易记录
- 报告内容写入页面正文
