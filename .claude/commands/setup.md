# 首次配置 / 更新配置

执行交易日志的首次配置或更新已有配置。

## 执行步骤

1. 读取 `references/setup-guide.md` 获取完整引导流程
2. 读取 `references/notion-schema.md` 获取默认 schema 参考

## 配置流程概要

按以下顺序逐步引导用户（对话式，不要一次问完所有问题）：

1. **语言版本** — 中文版还是英文版 Notion 模板
2. **获取数据库信息** — 用户提供 Notion 数据库链接或截图，使用 Notion MCP 工具 fetch schema
3. **字段映射** — 将用户的字段与默认 Ultra Trader Pro schema 对比，展示映射结果让用户确认
4. **识别 Relation 数据库** — 列出关联数据库的现有选项（账户、策略、品种、错误类型等）
5. **券商确认** — 用户主要使用哪个券商（影响截图识别准确率）
6. **复盘数据库** — 确认复盘日历数据库是否存在
7. **时区确认** — 默认美东时间
8. **保存配置** — 将配置信息写入项目根目录的 `trading-config.md`
9. **功能验证** — 建议用户做一次测试录入

## 配置存储

配置完成后，将所有信息写入项目根目录的 `trading-config.md`，格式参考 `references/setup-guide.md` 中的模板。
